# 如何创建vuepress站点

# 准备工作

vuepress是一套页面静态化的工具，基于vue，vue-router,vue ssr等技术。把markdown文件转化成html页面，在转化的过程中会生成导航和侧栏链接。让文档编写者能够更专注于文档内容。

这个工具是工作在node环境下的，所以首先得安装node。而且版本不能低于8.6。

另外vuepress默认主题是通过包管理工具来下载安装的。所以还得安装npm或者yarn。

我选择的是yarn。

# 创建vuepress站点

  • 创建项目目录

我的项目目录是mydocs，路径是D:\vue\mydocs

  • 进入目录创建
D:
cd vue/mydocs
  • 创建站点
yarn create vuepress

选择docs

输入相关信息，比如描述，邮箱，作者等信息

由于创建站点前初始化了一下，所以此时提示是否覆盖package.json。我选择了覆盖

  • 运行站点
vuepress dev docs

站点成功运行后，将显示站点地址
